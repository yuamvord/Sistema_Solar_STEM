HttpClient::getTlsStreamContextDefaults ( $cafile)
protected

CN_match and SNI_server_name are only known once a URL is passed. They will be set in the getOptionsForUrl() method which receives a URL.

cafile or capath can be overridden by passing in those options to constructor.

Attempt to find a local cafile or throw an exception. The user may go download one if this occurs.

Disable TLS compression to prevent CRIME attacks where supported.
