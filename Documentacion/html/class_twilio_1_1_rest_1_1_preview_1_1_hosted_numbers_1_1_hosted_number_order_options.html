string$accountSidThis defaults to the AccountSid of the authorization the user is using. This can be provided to specify a subaccount to add the HostedNumberOrder to.
string$friendlyNameA 64 character string that is a human readable text that describes this resource.
string$uniqueNameOptional. Provides a unique and addressable name to be assigned to this HostedNumberOrder, assigned by the developer, to be optionally used in addition to SID.
string[]$ccEmailsOptional. A list of emails that the LOA document for this HostedNumberOrder will be carbon copied to.
string$smsUrlThe URL that Twilio should request when somebody sends an SMS to the phone number. This will be copied onto the IncomingPhoneNumber resource.
string$smsMethodThe HTTP method that should be used to request the SmsUrl. Must be either GET or POST. This will be copied onto the IncomingPhoneNumber resource.
string$smsFallbackUrlA URL that Twilio will request if an error occurs requesting or executing the TwiML defined by SmsUrl. This will be copied onto the IncomingPhoneNumber resource.
string$smsFallbackMethodThe HTTP method that should be used to request the SmsFallbackUrl. Must be either GET or POST. This will be copied onto the IncomingPhoneNumber resource.
string$statusCallbackUrlOptional. The Status Callback URL attached to the IncomingPhoneNumber resource.
string$statusCallbackMethodOptional. The Status Callback Method attached to the IncomingPhoneNumber resource.
string$smsApplicationSidOptional. The 34 character sid of the application Twilio should use to handle SMS messages sent to this number. If a SmsApplicationSid is present, Twilio will ignore all of the SMS urls above and use those set on the application.
string$addressSidOptional. A 34 character string that uniquely identifies the Address resource that represents the address of the owner of this phone number.
string$emailOptional. Email of the owner of this phone number that is being hosted.
string$verificationType
string$verificationDocumentSidOptional. The unique sid identifier of the Identity Document that represents the document for verifying ownership of the number to be hosted. Required when VerificationType is phone-bill.
